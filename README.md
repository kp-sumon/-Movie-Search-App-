# 🎬 Movie Search App

A sleek, responsive, and user-friendly web application that allows users to search for their favorite movies and instantly view key details including posters, ratings, release years, and plots using the OMDb API.

## 🔥 Features
* **Live Movie Search:** Fetches real-time movie data directly from the OMDb server.
* **Rich Data Presentation:** Displays Movie Title, Release Year, IMDb Rating, and Runtime.
* **Modern UI/UX:** Designed with an attractive dark mode theme and a trendy glassmorphic navigation bar.
* **Smart Loader:** Includes a smooth spinner animation while fetching data from the API.
* **Fully Responsive:** Optimized for all screen sizes, including smartphones, tablets, and desktops.

**Tech Stack:** HTML5, Vanilla JavaScript, OMDb REST API, Tailwind CSS, FontAwesome
